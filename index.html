Mastermind

Anleitung

Dein Ziel ist es, die versteckte Farbkombination herauszufinden. Wähle als erstes vier Farben. Wenn du mit dem OK-Button deinen Versuch bestätigt hast, bekommst du auf der rechten Seite einen Hinweis, was an deinem Versuch stimmt.
Weiss steht dafür, dass du eine richtige Farbe gewählt hast, sie aber nicht am richtigen Platz ist.
Schwarz steht dafür, dass sowohl die Farbe als auch der Platz stimmt.
Verändere deine Farbkombination so lange, bis du entweder den Code geknackt hast oder deine Versuche aufgebraucht sind.

Tipp: Im Code können Farben mehrfach vorkommen!
